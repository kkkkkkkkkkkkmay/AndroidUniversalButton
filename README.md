# AndroidUniversalButton
众所周知，Android 自带的Button 一直都是一个鸡肋的存在，不仅丑，而且在设置背景图片后竟然没有自带按下的效果，要么自己准备图片，要么写xml，简直麻烦，比起iOS的 UIButton 差远了。因此顺手写了一个按下后自动变色的Button，在background为图片或纯色的情况下都有效。
